# Free Cron Expression Explainer API

Free cron expression explainer API - plain-English description plus next run times for any cron expression.

**Live tool & docs:** https://cron.wrapper-agency.com

## Quickstart - free API, no key required

```bash
curl "https://cron.wrapper-agency.com/api/v1/cron?expr=0%209%20*%20*%201-5"
```

Clean JSON, open CORS, no signup. Try the interactive tool and full docs at https://cron.wrapper-agency.com
